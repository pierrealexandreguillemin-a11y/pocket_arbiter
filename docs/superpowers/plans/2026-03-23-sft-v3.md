# SFT v3 Training Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Train SFT v3 (2 epochs LR 1e-5) on Kaggle T4 with ~10 checkpoints to find the sweet spot between v1 (over-learns) and v2 (under-learns).

**Architecture:** Fork train_sft_v2.py -> train_sft_v3.py with 3 config changes (epochs 1->2, save_total_limit 6->12, output paths v2->v3). LR stays 1e-5 (same as v2). Push to Kaggle, download outputs, select best checkpoint by moving-average loss analysis.

**Tech Stack:** Kaggle T4, TRL 0.16.0, Gemma 3 270M IT, TAPT checkpoint-88

**Spec:** docs/superpowers/specs/2026-03-23-sft-v3-design.md
